## 8. Conclusion

本論文では、Einstein-Cartan 接続に基づく Weyl 二乗結合 $\alpha C_{\rm EC}^2$ を導入した homogeneous minisuperspace を、 $S^3\times S^1$、 $T^3\times S^1$、 $Nil^3\times S^1$ の 3 トポロジーにわたって比較し、 EC-Weyl 拡張が homogeneous bulk geometric sector の mode dictionary に何を保持し、何を更新するかを整理した。

第一に、EC-Weyl 結合の新寄与は一般には任意の torsion 背景に現れるのではなく、AX / VT 背景では dropout し、MX 背景においてのみ本質的な修正が現れることを確認した。  
同時に、Palatini 型の非伝播性は EC 拡張後も保持され、torsion sector は kinetic mode としては立ち上がらない。  
このことにより、LC 版で確立された homogeneous sector の基本構造は、EC 接続下でも大きくは崩れないことが明らかになった。

第二に、 $S^3\times S^1$ については、paper03 で確立した homogeneous mode dictionary を基底として、EC-Weyl 拡張下でどの部分が不変であり、どの部分に修正が入るかを明示した。  
本論文の立場では、 $S^3$ は依然として最も豊かな homogeneous bulk geometry を与える基準トポロジーであり、EC 拡張はその辞書を破壊するのではなく、更新則を与えるものとして位置づけられる。

第三に、 $T^3\times S^1$ については、mode dictionary は本質的に null / trivial structure として閉じる。  
すなわち、EC-Weyl 拡張を行っても、 $T^3$ は homogeneous bulk sector に新たな nontrivial mass structure を与えず、cross-term の一部も純粋に kinematic な起源として理解される。  
この意味で、 $T^3$ は「何が起きないか」を明示する基準背景として機能する。

第四に、 $Nil^3\times S^1$ については、EC-Weyl 拡張により topology-specific な false vacuum と anisotropic split structure が現れ、spin-2 quintet の分裂と spin-1 の 1 軸質量分裂が幾何学的に固定されることを示した。  
したがって $Nil^3$ は、LC 版では見えなかった EC 固有の幾何学的相構造を担う背景として位置づけられる。

以上を総合すると、本論文は、 **axial + vector-trace torsion ansatz の範囲において、EC-Weyl 下の homogeneous three-topology bulk sector を閉じる** ものである。  
ここで閉じたのは、あくまで共通 ansatz に基づく比較可能な homogeneous geometric sector であり、この枠組みの上で $S^3$、 $T^3$、 $Nil^3$ の役割分担が明確化された。

本論文の結果は、adopted homogeneous torsion ansatz の範囲で、以後の非均質セクターや境界応答への理論展開の基礎を与える。
